# OpenFisca Dash UI

UI for OpenFisca using [Python Dash](https://dash.plot.ly/).

## View on Heroku

https://openfisca-dash-ui.herokuapp.com/

## Install

```bash
mkvirtualenv openfisca-dash-ui
pip install -r requirements.txt
```

## Start server

```bash
python app.py
```

Open http://localhost:7777/
